## 📌 Introduction

Welcome to the **C Learning & Practice Project** repository. This workspace contains a variety of C modules, starting from simple numeric base converters and mathematical algorithms to console-based games and local financial calculation utilities.

All practice modules are located under the **[`C Practice`](<C Practice/>)** directory. Each project features its own source code and a dedicated README with compilation/run instructions.

---

## 📂 Repository Structure

Below is the layout of the project, highlighting each practice module:

```text
.
├── C Practice/
│   ├── Banknote type counting system/               # Greedy algorithm banknote calculator
│   ├── Calculator Vector System (Multiple, Norm)/  # Vector dot products & Euclidean norms
│   ├── Calculator Vector System (Sum, Sub, Scaler)/ # Vector addition, subtraction & linear combinations
│   ├── Cambodia payroll tax system/                 # Tax calculations and progressive brackets
│   ├── Convert Binary to Decimal/                   # Binary to decimal integer conversion
│   ├── Convert Binary to Octal/                     # Binary to base-8 converter
│   ├── Convert Decimal to Binary/                   # Decimal to base-2 converter
│   ├── Convert Decimal to Hexadecimal/              # Decimal to base-16 converter
│   ├── Convert Decimal to Octal/                    # Decimal to base-8 converter
│   ├── Convert Octal to Decimal/                    # Octal to decimal converter
│   ├── Covert Octal to Binary/                      # Octal to binary converter
│   ├── Get the C Version in use/                    # Queries compiler metadata and C standards
│   ├── Khmer numeral system for calling money/       # Converts numeric currency to spoken Khmer terminology
│   ├── Kla Klok/                                    # Simulated Khmer traditional dice game
│   ├── Print name, DOB, and mobile number/          # String formatting options (simple, right-aligned, zero-padded)
│   └── Reverse the number/                          # Digit reversal utility
├── Print Block F and a large C/                      # Graphical F and C block printing programs
├── .gitignore                                       # Git ignore rules for compiled binaries (.exe, etc.)
└── README.md                                        # This main navigation guide
```
