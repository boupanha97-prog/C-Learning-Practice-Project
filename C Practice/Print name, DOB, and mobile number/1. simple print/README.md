# 🖨️ Simple Print

This program uses literal string arguments with the basic standard `printf()` function to dump static personal details to the screen.

## 📄 Source File
- [`simple.c`](simple.c)

## 🚀 Compilation & Execution
```powershell
gcc simple.c -o simple
.\simple
```

## 🔍 Code Explanation
Uses standard standard output (`printf`) to display name, date of birth, and mobile number in a simple format.
