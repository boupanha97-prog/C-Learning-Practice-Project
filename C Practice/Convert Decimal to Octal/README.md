# 🔢 Decimal to Octal Converter

Converts base-10 decimal numbers into base-8 octal representation.

## 📄 Source File
- [`dec_to_octal.c`](dec_to_octal.c)

## 🚀 Compilation & Execution
```powershell
gcc dec_to_octal.c -o dec_to_octal
.\dec_to_octal
```

## 📊 Expected Output Results
```text
Input Dec: 83
---------------
Output Bin: 123
```
